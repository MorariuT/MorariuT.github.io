# Getting Started

Welcome to the AI/ML Programmers Handbook! This guide will help you navigate the documentation and get the most out of our resources.

## What is This?

This handbook is an alternative to [cp-algorithms.com](https://cp-algorithms.com/index.html) but specifically for Artificial Intelligence and Machine Learning. All information is based on scientific articles and personal research, with proper citations throughout.

## Navigation Structure

The documentation is organized into the following sections:

### **Basics**
Start here if you're new to AI/ML! We cover:
- Matrix Algebra - Essential linear algebra concepts
- Statistics - Probability and statistical foundations
- Cost/Loss Functions - Understanding different loss functions
- Gradient Descent - How optimization works

### **Linear Models**
Build your foundation with fundamental algorithms:
- Linear Regression - Predicting continuous values
- Logistic Regression - Classification fundamentals

### **Neural Networks**
Dive deep into neural networks:
- Basics - Architecture and components
- Forward Propagation - How data flows through networks
- Back Propagation - Training algorithms
- Simple Neural Network - Hands-on implementation

### **Computer Vision**
Explore image processing and analysis:
- Basics - Computer vision fundamentals
- Convolution - Understanding convolutional operations
- CNN - Convolutional Neural Networks
- Data Processing - Image preprocessing techniques

### **Natural Language Processing**
Process and understand text:
- Basics - NLP fundamentals
- Simple Embedders - Word and sentence representations
- Attention - Attention mechanisms
- Transformers - Modern transformer architectures

### **Ensemble Models**
Learn about combining multiple models:
- Decision Trees - Tree-based learning
- Random Forests - Ensemble tree methods
- Boosting - Sequential ensemble methods
- Voting Models - Model combination strategies

### **Data Analysis**
Explore data analysis techniques:
- Introduction - Data analysis fundamentals
- PCA - Principal Component Analysis
- LDA - Linear Discriminant Analysis
- UMAP - Dimensionality reduction visualization
- Statistics - Statistical analysis methods
- Useful Plots - Visualization techniques

## How to Use This Handbook

2. **If you know the fundamentals**: Jump to the specific topic you're interested in
