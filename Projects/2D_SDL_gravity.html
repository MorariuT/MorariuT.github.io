Explanation

In this section I will (try to) explain the code.

This function is doing all the hard work. It takes as arguments the main object (aka. the class parent) and another object and then it calculates the attraction using Newton's law of universal gravitation.

                

        std::pair Object::calculate_gravity(const Object &other)
        {
            const float G = 5;
            float dx = other.x - this->x;
            float dy = other.y - this->y;
            float distance = get_distance(this->x, this->y, other.x, other.y);

            if (distance == 0) return {0, 0};

            float force = G * (this->mass * other.mass) / (distance * distance);
            float ax = force * (dx / distance) / this->mass; 
            float ay = force * (dy / distance) / this->mass;

            return {ax, ay};
        }
